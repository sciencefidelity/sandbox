# Dart

To run the project `dart run`
To run the tests `dart test`

```shell
# run project
dart run

# start the app
# fswatch -o -1 -d bin | (while read; do dart run; done)
dartwatch

# run the tests
dart test

# start the test suite
# fswatch -o -1 -d bin test | (while read; do dart test; done)
darttdd

# format code
dart format
```
